## [1.0.0](https://github.com/yuyash/md2cv-vscode/releases/tag/v1.0.0) (Initial Release)

### Features

- Real-time preview of CV/Resume from Markdown
- PDF export functionality
- Support for Western-style CV and Japanese formats (履歴書/職務経歴書)
- IntelliSense support for md2cv frontmatter
- Multiple paper size support (A3, A4, B4, B5, Letter)
- Photo insertion support
- Synchronized scrolling between editor and preview
- Template generation for quick start
- Multi-language support (English/Japanese)
